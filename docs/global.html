diverge(…functions) → {function}

Returns a function that accept an argument which will be applied to every function in parameter and given back as an array AKA. Parallelized compose.
Parameters:
Name Type Attributes Description

Example
let fn1 = compose (uppercase,prop('key'))
let fn2 = compose (reverse,prop('another_key'))
let fn3 = diverge (fn1,fn2);
fn3 ( {'key':'foo', 'another_key':'bar'})
//  ['rab','FOO']

diverge (fnA,fnB)(x) == [fnA(x),fnB(x)]
